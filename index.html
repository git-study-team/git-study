使い方

git add -A …　ステージングにファイルをあげる
git commit -m …　"コメントを書く"
git push origin ブランチ名 …　リモートリポジトリにアップ
git pull origin ブランチ名 …　リモートリポジトリのデータをダウンロード

スタッシュ編

git stash save …　変更を退避する git stash list … 退避リスト一覧を表示
stash@{0}: WIP on master: 57dd867 Merge pull request #32 from okutani-t/use-git2
スタッシュ名 : WIP on ブランチ名: HEADのメッセージ git stash apply stash@{0} … 退避した作業を戻す
git stash drop stash@{0} …　stash listに残ったログを消す git stash pop stash@{0} …　上記の２つの作業を同時に行う(作業を戻す& ログを消す)
